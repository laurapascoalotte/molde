Sistema Boletim Fácil


Aqui apresentaremos uma breve orientação da nossa página inicial para acessar nosso Sistema Boletim Fácil, que foi criado para facilitar a vida de alunos e professores no acesso do boletim escolar através da Internet.

O sistema Boletim Fácil é disponibilizado via WEB, portanto para acessá-lo, basta digitar o endereço abaixo:

https://www.boletimfacil.com.br/

Os usuários, tanto professores e alunos, serão direcionados para a tela inicial com algumas opções para escolha:

  • Login: ao selecionar qual a Escola desejada, inserir usuário e senha e clicar no botão Entrar, acessa o sistema para utilização.
  • Cadastro: ao selecionar essa opção, irá ser apresentado um formulário para cadastro de um novo usuário para utilizar o sistema.
  • Ajuda: aqui o usuário será redirecionado para a página de ajuda, com perguntas mais frequentes e suas respostas, em relação ao acesso e utilização do sistema.
  • Escola: aqui temos um menu drop-down com a escolas que utilizam o nosso sistema. O usuário deverá escolher qual a Unidade de Ensino que deseja acessar.
  • Usuário: aqui você deverá digitar o nome de usuário previamente cadastrado.
  • Senha: nesse campo deverá ser digitada a senha cadastrada.
  • Entrar: após selecionar e escola desejada, digitar o usuário e senha, clique nesse botão para acessar o sistema.
  • Limpar: caso digite errado alguma informação durante o login, clique nesse botão para limpar os dados e inseri-los novamente.
  • Esqueci a senha: aqui o usuário tem a possibilidade de recuperar o seu acesso, caso tenha esquecido seu usuário/senha.
